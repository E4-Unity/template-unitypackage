# 유니티 패키지 템플릿

## 권장 사항

유니티 패키지 설치를 위한 `git URL` 을 기억하기 쉽게 만들기 위해 GitHub 저장소 이름은 유니티 패키지 이름으로 설정

- [Git URL을 통한 설치 - Unity 매뉴얼](https://docs.unity3d.com/kr/2021.3/Manual/upm-ui-giturl.html)
